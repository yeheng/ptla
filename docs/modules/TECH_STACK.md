# 技术栈说明

## 1. 前端技术栈

- **Tauri**: 用于构建跨平台桌面应用程序的框架
- **Vue 3**: 前端框架，使用Composition API
- **TypeScript**: 类型安全的JavaScript超集
- **ShadcnUI**: 基于Tailwind CSS的UI组件库
- **Vite**: 前端构建工具

## 2. 后端技术栈

- **Rust**: 系统编程语言
- **Actix-web**: 高性能Web框架
- **SQLx**: 异步SQL查询构建器
- **SQLite**: 轻量级关系型数据库
