# 系统架构设计

## 1. 整体架构

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│                 │     │                 │     │                 │
│   Vue Frontend  │◄───►│  Tauri Bridge  │◄───►│  Rust Backend   │
│                 │     │                 │     │                 │
└─────────────────┘     └─────────────────┘     └─────────────────┘
                                                         │
                                                         ▼
                                                 ┌─────────────────┐
                                                 │                 │
                                                 │    SQLite DB    │
                                                 │                 │
                                                 └─────────────────┘
```

## 2. 组件职责

1. **前端层 (Vue)**
   - 用户界面渲染
   - 状态管理
   - 用户交互处理
   - 与Tauri桥接层通信

2. **Tauri桥接层**
   - 提供系统API访问
   - 处理前端与后端的通信
   - 管理窗口和系统资源

3. **后端层 (Actix-web)**
   - 业务逻辑处理
   - 数据持久化
   - API端点提供
   - 数据验证和安全性

4. **数据层 (SQLx + SQLite)**
   - 数据存储
   - 数据查询和操作
   - 数据迁移管理
